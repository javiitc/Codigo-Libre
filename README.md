# Codigo-Libre
QUE HACE EL CODIGO? 
Con este codigo he hecho un Quiz sobre dos temas a elegir, peliculas o videojuegos.
Luego de elegir tema, es cuestión de seleccionar la pregunta que se quiere responder primero, seguido de la elección de la segunda pregunta  y finalmente se responde a la ultima pregunta pasando automaticamente al recuento de preguntas correctas.
También esta pensado para que no se pueda seleccionar más allá de 1 o 2 (cuando seleccionas tema) o de 1 o 3 (en caso de las preguntas).
